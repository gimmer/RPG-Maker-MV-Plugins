## Gimmer_FirstFinds

Allows you to set a text string or common event that fires
when you first loot an item (either on the map or in battle)

#### Text Tutorial:

Simply set a note tag with either:

<ff:text you want to show up when you first get an item>
or
\<ffe:[commontEventId]> for a common event that you want to fire


Plugin parameters let you choose what kind of window will popup will appear if you use the text option
