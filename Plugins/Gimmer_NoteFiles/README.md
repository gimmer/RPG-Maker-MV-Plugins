### Gimmer_NoteFiles

Instead of putting note tags into the notes folder, just put the following:

<noteFile:path/to/file.txt>

Place the file in the data folder. You can add sub folders if you want.

E.G. <noteFile:notes/harold.txt>

The files will just contain whatever you would have put in a box. The system will try to cache files to avoid loading them repeatedly (mostly map events, between map loads)
